GYM FIT FITNESS CENTER — WEBSITE PACKAGE
==========================================

WHAT'S INSIDE
  index.html   -> the full website (one page)
  style.css    -> all styling
  script.js    -> mobile menu, membership price toggle, animated stats
  assets/      -> gym photo used in the hero section

HOW TO PREVIEW IT
  Just double-click index.html — it opens in any browser.
  For the best experience (especially the Google Map at the bottom),
  upload it to real hosting instead of opening the file directly.

HOW TO PUT IT LIVE (pick one)
  1. Netlify / Vercel (free): drag the whole folder onto netlify.com/drop
     and it's live in ~30 seconds.
  2. Any shared hosting (Hostinger, GoDaddy, etc.): upload all files via
     File Manager or FTP into the public_html folder.
  3. Buy a domain like gymfitsus.in or gymfitfitnesscenter.com and point
     it at whichever hosting you choose.

WHAT YOU CAN EDIT YOURSELF (no coding needed)
  - Phone number: search-and-replace "9861661717" in index.html
  - Prices: edit the numbers inside the "plan-card" sections in index.html
  - Reviews: edit/add cards inside the "review-track" section
  - Hero photo: replace assets/gym-hero.jpg with a new photo (keep the
    same filename, or update the src path in index.html)
